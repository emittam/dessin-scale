# dessin-scale

# Features
This script is draw grid image

# Requirement
* Pillow 8.2.0

# Installation

```bash
pip install git+https://github.com/emittam/dessin-scale
```

# Usage

```bash
dessin-scale -g [縦横のグリッド数] [画像のPath]
```
